## 🚀 Швидкий старт

### 1️⃣ Запуск Backend
```bash
cd app
python -m uvicorn main:app --reload --port 8000
```

### 2️⃣ Запуск Frontend
```bash
cd frontend
npm install  # перший раз
npm run dev
```

### 3️⃣ Вхід як адмін
- Локально можна використовувати:
  - **Логін**: `admin`
  - **Пароль**: `1234`
- Або через Steam (налаштовано на фронтенді)

### 4️⃣ Доступ до Адмін Панелі
- Перейдіть на `http://localhost:5173`
- Натисніть \"⚙️ Адмін\" в навігації (видима для залогінених користувачів)

### 5️⃣ Управління адміністраторами (NEW!)
```bash
# Отримати токен
TOKEN=$(curl -X POST "http://127.0.0.1:8000/token" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "username=admin&password=1234" | jq -r '.access_token')

# Дати адміну права по steam_id
curl -X POST "http://127.0.0.1:8000/admin/users/123456789/make-admin" \
  -H "Authorization: Bearer $TOKEN"

# Переглянути всіх адмінів
curl -X GET "http://127.0.0.1:8000/admin/admins" \
  -H "Authorization: Bearer $TOKEN"
```

## 🔄 Робочий процес

### Додавання новини
1. Заповніть форму (заголовок, вміст, URL зображення)
2. Натисніть \"Додати\"
3. Новина з'явиться у списку

### Редагування новини
1. Натисніть \"✏️ Редагувати\" на потрібній новині
2. Внесіть зміни у формі
3. Натисніть \"Зберегти\"

### Видалення новини
1. Натисніть \"🗑️ Видалити\"
2. Підтвердіть дію
3. Новина видалиться

## 📚 API Endpoints

```
GET  /news/                    # Отримати всі новини
GET  /news/{news_id}           # Отримати одну новину
POST /news/                    # Створити (адмін)
PUT  /news/{news_id}           # Оновити (адмін)
DELETE /news/{news_id}         # Видалити (адмін)
```
